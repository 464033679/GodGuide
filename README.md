# GodGuide

#### 介绍
Cocos Creator 新手引导框架，支持浏览器自动引导，流程录制与回放，可实现自动化测试！

#### 使用说明

1. 使用Cocos Creator 2.0.10 打开项目
2. 引导相关代码在assets/GodGuide目录下
3. Task1.js 、Task2.js 为引导任务配置
